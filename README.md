# NChampions.WebApi
.Net 5.0 project based on Medium article written by Carlos Eduardo de Paula Soares (https://edupaulasoares.medium.com/net-5-0-webapi-de-ponta-a-ponta-f455832e84b0)

In this project, you'll find:
- .Net Core 5.0
- Docker
- Mediator
- FluentValidation
- SQL Server
- Dapper
- Entity Framework
